### 环境搭建

1. 全局安装 vue-cli
```
npm i -g vue-cli
```

2. 创建vue项目
```
// [project-name] 替换为要创建的项目名
vue create [project-name]
```

3. 使用 vue-cli 插件 vue-cli-plugin-electron-builder 给项目添加 electron 相关代码
```
vue add electron-builder
```

-- vue-cli3 + electron 环境搭建完毕
